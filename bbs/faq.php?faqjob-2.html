注册与会员登录
关于注册
您需要 注册成为会员 才能正常使用论坛的各项功能，比如：
  • 发布一个主题
  • 回复其他会员的帖子
  • 编辑自己的帖子
  • 查看其他会员信息,排行榜等
  • 给其他会员发送短信息
  • 搜索您所需要的资料
    等等 .....注册时，您需要一个用户名、密码和一个有效的email地址。输入您的email地址不会给您带来'垃圾邮件'，您可以在注册时选择隐藏email地址。
    • 关于登录论坛
    如果您以前是该论坛的会员，请点击 登录 进入登录界面，填写用户名和密码。
    同时您可以选择是否隐身登录，如果管理员允许您所处的用户组拥有此权利，这样别人将无法知道您是否在线。
    Cookie 有效期介绍: 主要用于在您离开一会儿或者重新进入一个主题或者帖子的时候确认您的身份。当您登录时， 您将有一个'cookie 有效期:'选项，您可以选择cookie保存的时间，这时系统将您的用户名和密码用cookie保存，储存在您的电脑。如果您是使用一台共用的电脑，比如在图书馆、网吧，或者其他不能完全信任的地方， 建议使用'即时'选项，以免您的帐号被他人盗用。
    找回密码
    如果您之前是会员而忘记了密码，可用此功能 找回密码(邮箱必须有效)，在取回密码页面输入您注册时候的用户名，系统将自动发送密码到您的有效邮箱中。
    会员控制面板
    关于会员核心数据
    在 编辑会员核心数据 时，考虑到安全性必须输入原密码，同时您也可以更改现有的密码，同时确认您的email地址是有效的，必要时系统将会发送一些论坛信息至您的邮箱，比如密码发送等。注意不能更改用户名，如有特殊情况需要修改用户名，请与管理员联系。
    关于个性化签名
    注册时，可以设置自己的个性签名。您的签名将出现在每个帖子底部，您也可以选择不显示个性签名，在发帖和回复的时候不选中"使用签名"选项。在编辑个性化签名时您可以使用 Wind Code 进行编辑多彩的签名。注意签名的字节数是跟您的等级相关的，所以您的签名长度是受限制的!
    关于自定义头像
    个人头像会出现在每个帖子中用户名的下方，头像可以是由论坛提供或者您自己输入完整的 URL 或上传 均可以，您可以选择一个最能代表您个性的头像。如果论坛允许使用自定义会员头像， 您可以使用URL 地址的头像或自己上传头像。
    论坛发帖与相关权限
    关于会员
    该积分用以判定一个会员对论坛宣传上的! 如果您使用 论坛的URL ?a= 自己的论坛数字ID 给您的一位朋友，使他们登录到论坛里，您的将增加 1 单位。举例: http://www.phpwind.net/?u=3558 论坛数字ID 可在 控制面板 --> 查看个人资料 里查阅
    关于会员等级提升
    一般是以您的发帖数来划分等级的，只要您多参与论坛讨论。等你的发帖数超过一定的数量时，您的等级将提升(有些论坛采用其他积分划分等级)。当然您同时还可以向管理员申请其他的特殊用户!
    关于论坛权限
    如果您看到论坛的某些版块为认证版块，说明您现在的用户组还没有权限进入。
    关于发帖的 Wind Code
    可以使用Wind Code 编辑您的帖子、短消息、签名，您可以设置帖子文字的字体、颜色等格式。Wind Code 代码使用简单。
    关于投票
    这项功能允许您提出一个问题，并设置一些可能的答案。其他会员则选答案参与投票，投票结果将在主题上显示。投票的例子见：
      什么是您最喜欢的城市? ( 在编辑框中 以 换行的 形式写入以下各项 )
    1. 温州
    2. 杭州
    3. 北京
    4. 上海
    5. 深圳
    　其他会员可以参与投票，您还可以设置投票是否允许多选，及多选的个数。每个会员只有一次投票机会，投票后可编辑投票！
    关于附件
    如果论坛允许使用上传附件的功能， 您可以在帖子中上传一些文件.。可以是一个图片、一个文本、一个压缩文件等. 您上传的每种文件都有大小限制( 最大为: 2 M )，论坛跟您的硬盘所采用的文件扩展名并不一致!上传一个附件，只有下载到自己的电脑上才能浏览。论坛限定了上传文件的类型:可以上传的文件扩展名: gif jpg rar txt zip swf jqf 发帖完毕后，附件将显示在您所发布的信息中.
    编辑与删除帖子
    注册会员可以编辑或删除自己的帖子，点按钮来管理帖子。管理员可以编辑和删除论坛的所有帖子，版主只能编辑和删除指定版块内的帖子。
    关于版主
    版主负责论坛中某个版面。可以修改，删除或移动他们所管理的版面中的任何帖子等权限。任命某个版面的版主，通常是对本版有特殊贡献或对某领域有特殊才华的会员的一种奖励。
    WAP 说明
    
测量控制与仪器仪表工程师论坛 目前已开通支持 WAP (手机移动论坛)功能.

手机访问地址：http://www.mciea.org/bbs/wap/

说明：只要您开通了移动GPRS或联通CDMA的数据服务，在支持WAP的手机上即可通过此网址访问 测量控制与仪器仪表工程师论坛

手机资费说明：移动GPRS或联通CDMA的数据服务包月或流量费用，无需任何额外的费用支出

主要应用范围：

    通过手机您可以随时随地的浏览您在网络论坛上的活动信息，了解您发表的帖子的阅读、回复情况。
