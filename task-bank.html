What is the MCIEA Task Bank?

​The MCIEA Task Bank is a public online resource that features performance-based tasks created by educators in MCIEA districts.
Within the public task bank, each task is represented as a card that contains various details about the tasks as well as resources used for administering the tasks. These tasks can be searched using filters found to the left of the Tasks.

Each filter type will display a list of all possible search values when selected. These options can be browsed and selected or begin typing the value in the search box to get to a specific filter type more quickly.

Additionally, the entire collection can be browsed by scrolling down in your browser and clicking on the “load more” button to add more task cards to your page.

MCIEA Task Bank Instructions 

We have created an in-depth instruction packet to help guide you through the online task bank. 
​
It not only provides step-by-step explanations and details on accessing and utilizing the tasks currently available in the public task bank, but also provides information on how to prepare and submit performance assessment tasks that you have created.
View the instruction packet
Submit your Performance-Based Assessment Task to the MCIEA Task Bank and share with the entire Consortium.

Share your Performance-Based tasks with teachers in other districts so that they can learn from your work and experience.  Each author earns ten PDPs upon final submission of their performance task.
 
Steps for final task submission:
  • Prepare the following three pieces of the task as well as any other resources
    • MCIEA Task Template: Walks you through the task design process step-by-step.
    • MCIEA Rubric: Provides a starting point for building out a rubric to complement your performance task.
    • Student Instructions
  • Fill out the task submission form
  • Task will be reviewed by two MCIEA teachers
    • Once task is reviewed (2-3 weeks), reviewers will send author(s) their comments and recommendations
  • Revise task based on comments and recommendations, and resubmit in this form as a revised task
  • After task is final, you will receive a PDP certificate for your work on the task equal to 10 hours; PDPs will only be provided if you are able to submit a revised task based on comments and recommendations from reviewers
  • Finalized tasks will be uploaded to the public task bank
